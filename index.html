street_lite

street_lite is an application that keeps users up-to-date with public events held on street closures in NYC. It utilizes the power of NYC Open Data API and Google Maps API to provide users users with the information they need to attend an event near them. Built in collaboration with Harjit Singh, Trey Pisano and Arvid Hossain

Slurp

Based on my love for coffee, Slurp is a full-stack clone of Yelp, with a focus on cafes, built from the ground up using a Rails backend and a React frontend. It allows users to view cafes, create an account, leave reviews, update them and delete them.

Audio Atlas

The first project I worked on, Audio Atlas is a JavaScript data visualizer that uses the Spotify API as well as D3 to allow users to enter the name of an artist and get a collpsible tree chart containing the entire discography of their albums and songs off those albums.
